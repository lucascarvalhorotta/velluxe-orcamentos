# Velluxe Decor - Sistema de Orçamentos v2.0

Sistema de orçamentos com autenticação e banco de dados Supabase.

## Features
- ✅ Login com email/senha
- ✅ Todos usuários veem todos orçamentos
- ✅ Salvar no Supabase (cloud)
- ✅ PDF profissional
- ✅ Catálogo de produtos
